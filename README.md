# Ilvet Farm — site de prezentare

Site static, responsive, realizat pentru cabinetul veterinar Ilvet Farm din Craiova.

## Pornire locală

Din acest folder rulează:

```powershell
python -m http.server 4173
```

Apoi deschide `http://127.0.0.1:4173`.

Nu sunt necesare dependențe sau proces de build.
